---
title: Platform Overview
description: The shortest useful mental model for the Wacht product surfaces.
---

Wacht is split across distinct surfaces:

- `console` for operators
- `platform-api` for control-plane and infrastructure services
- `frontend-api` for customer-facing runtime APIs
- `react-sdk`, `node`, and `wacht-rs` for integrations

This docs layout is organized around that split so readers can quickly tell whether they are looking at product concepts, integration guides, or exact API contracts.
