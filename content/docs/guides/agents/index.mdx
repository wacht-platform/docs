# Agents

Most agent configuration lives at the deployment level — provider keys, default models, sandbox storage. Per-agent settings let you override or extend that for specific agents.

- [Model overrides](/docs/guides/agents/model-overrides) — pick a different strong or weak model for one agent without touching the deployment defaults.
- [Execution hooks](/docs/guides/agents/hooks) — run a sequence of tools when an execution starts and when it ends.
