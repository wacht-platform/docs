Rust backend APIs are framework-agnostic, and the SDK also ships framework middleware behind feature flags.

Start with [Axum](./axum), which is supported through the `axum` feature.

Axum guidance is split into focused pages:

- [Overview](./axum)
- [Auth Layer Setup](./axum/auth-layer-setup)
- [Extractors and Permissions](./axum/extractors-and-permissions)
- [Gateway Authorization](./axum/gateway-authorization)
