## Getting started

If you are evaluating the platform, start with [Product](/docs/product).

If you are integrating an application, go to [SDKs](/docs/sdks).

If you are implementing a full workflow, go to [Guides](/docs/guides).

If you need exact contracts, use [API Reference](/docs/reference).

## Core surfaces

### Console

The operator UI used to configure projects, deployments, authentication, billing, API authentication, webhooks, and AI features.

### Platform API

The Rust control plane, gateway, OAuth services, and worker stack used by operators and platform infrastructure.

### Frontend API

The customer-facing runtime API for sessions, organizations, workspaces, notifications, and end-user auth flows.

### SDKs

Frontend and server integrations layered on top of the runtime surfaces.
