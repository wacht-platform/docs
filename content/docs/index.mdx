Wacht is open source infrastructure for AI-native apps. Identity, organizations, machine auth, webhooks, notifications, and an agent runtime — built as one product on one model, instead of six vendors stitched together.

New here? Start with the **Start** section. It covers what Wacht is, the anatomy of a Wacht app, the mental model for designing one, and a working end-to-end build.
